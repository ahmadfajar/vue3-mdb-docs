# Grid

::: lead
CSS classes to build flexbox grid layout with twelve column system and six default responsive tiers.
:::

<SmallNote color="teal">Since v2.2.0</SmallNote>

## Overview 

The grid system is cloned from CSS [Bootstrap Grid](https://getbootstrap.com/docs/5.3/layout/grid/). 
More details information can be found at [Bootstrap Grid](https://getbootstrap.com/docs/5.3/layout/grid/) documentation. 


## Examples 

::: BlockHtml

```html
<div class="w-full px-3 text-center">
  <div class="row bg-blue text-light">
    <div class="col p-2 border border-default-color">col</div>
    <div class="col p-2 border border-default-color">col</div>
    <div class="col p-2 border border-default-color">col</div>
    <div class="col p-2 border border-default-color">col</div>
  </div>
  <div class="row bg-blue text-light mt-3">
    <div class="col-8 p-2 border border-default-color">col-8</div>
    <div class="col-4 p-2 border border-default-color">col-4</div>
  </div>
</div>

```
:::
