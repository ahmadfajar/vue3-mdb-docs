# Spacing

::: lead
CSS classes to quickly control the margin and padding of an element.
:::

<SmallNote color="teal">Since v2.2.0</SmallNote>


## Margin {class="mb-3"}

<div class="doc-api-reference mt-3">
<div class="doc-table-responsive table:font-mono:col-2 table:code:col-2">

| Class   | Styles |
|---------|--------|
| mx-auto | margin-left: auto; <div>margin-right: auto;</div> |
| mx-0  | margin-left: 0; <div>margin-right: 0;</div> |
| my-0  | margin-top: 0; <div>margin-bottom: 0;</div> |
| ms-auto  | margin-left: auto; |
| ms-0  | margin-left: 0; |
| me-auto  | margin-right: auto; |
| me-0  | margin-right: 0; |
| mt-0  | margin-top: 0; |
| mb-0  | margin-bottom: 0; |

</div>
</div>

## Padding {class="mb-3"}

<div class="doc-api-reference mt-3">
<div class="doc-table-responsive table:font-mono:col-2 table:code:col-2">

| Class   | Styles |
|---------|--------|
| px-0  | padding-left: 0; <div>padding-right: 0;</div> |
| py-0  | padding-top: 0; <div>padding-bottom: 0;</div> |
| ps-0  | padding-left: 0; |
| pe-0  | padding-right: 0; |
| pt-0  | padding-top: 0; |
| pb-0  | padding-bottom: 0; |

</div>
</div>
